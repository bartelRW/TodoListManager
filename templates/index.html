Todo-Listen-Verwaltung

Entwurf und Implementierung einer REST-Schnittstelle

Die Todo-Listen-Verwaltung ist eine Beispiel-Anwendung für die Realisierung einer REST-Schnittstelle.
Es ist möglich anhand folgender Endpunkte Listen mit entsprechenden Einträgen zu erstellen und bearbeiten.

GET
/todo-list/{list_id}
Liefert eine Liste
DELETE
/todo-list/{list_id}
Löscht eine Liste
POST
/todo-list
Fügt eine neue Liste hinzu
POST
/todo-list/{list_id}/entry
Fügt einen neuen Eintrag einer bestehenden Liste hinzu
PUT
/todo-list/{list_id}/entry/{entry_id}
Aktualisiert einen bestehenden Eintrag aus einer Liste
DELETE
/todo-list/{list_id}/entry/{entry_id}
Löscht einen bestehenden Eintrag aus einer Liste
GET
/todo-list/{list_id}/entry/entries
Liefert alle Einträge einer bestehenden Liste
GET
/todo-lists
Liefert alle Listen
